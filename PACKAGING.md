# Distribution Notes

## For Students

1. Unzip the package to a normal folder, such as Desktop.
2. Make sure desktop Microsoft Word is installed.
3. If this is the first time using the tool, double-click `Install-Dependencies.bat`.
4. Double-click `Run-WordReportTool.bat`.
5. Select a `.docx` report and wait for the output file.

The tool writes a new file next to the selected document, for example:

```text
report.with-directories.docx
```

The original document is not overwritten.

## Requirements

- Windows
- Desktop Microsoft Word
- Python 3
- Internet access during first-time dependency installation

If Python is missing, install it from <https://www.python.org/downloads/> and tick `Add python.exe to PATH`.
